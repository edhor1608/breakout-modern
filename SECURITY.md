# Security Policy

## Reporting A Vulnerability

Please do not open a public issue for security vulnerabilities.

Report vulnerabilities privately through GitHub Security Advisories for this repository:

https://github.com/edhor1608/breakout-modern/security/advisories/new

Include a short description, reproduction steps, affected versions or URLs, and any suggested fix if you have one.

## Scope

This is a static browser game. Security reports most likely involve dependency vulnerabilities, unsafe asset handling, deployment configuration, or browser storage behavior.
